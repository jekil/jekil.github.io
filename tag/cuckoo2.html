Such Cuckoo, much sandbox: dogeon report

Posted on June 08, 2014 in Tools • Tagged with cuckoo, doge • 2 min read

Several days ago I discovered one of the best projects of this year: dogeon.

DSON (Doge Serialized Object Notation) is a data-interchange format, that is easy to read and write for Shiba Inu dogs. It is easy for machines to parse and generate. It is designed to be as similiar as possible to the `DogeScript Programming Language <https://github.com/remixz/dogescript>`__. DSON is a text format that is not language independent but uses conventions that are familiar to a wide variety of japanese dog breeds.
